# Data Audit Notes

`download_log.csv` records how each raw dataset was collected for this VP35 research project. Each row corresponds to one species-specific dataset and should be updated when new raw downloads are added.

Raw files should never be manually edited. Keep the raw FASTA and CSV files unchanged so the filtering and audit steps remain reproducible.

Zaire is the primary EBOV dataset for within-Zaire VP35 variation analysis. Sudan, Bundibugyo, Taï Forest, Reston, and Bombali are expansion datasets for cross-species VP35 conservation analysis.

Across species, amino-acid differences should be described as homologous differences, not mutation frequency.
